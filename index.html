Projects

SMS Budget App with Spending Analysis

This application utilizes my reciept scanning component while also connecting with the Notion API to use as a frontend and as a databse. This app adds non-native functionality to a Notion page which allows for the construction of data visualization charts from Notion database data and replacement of page images in-place. Expense data is entered in the form of SMS messages and/or an MMS reciept image sent directly to a Twilio number and parsed according to totals. This project is ongoing.

Serverless MMS Reciept Scanner

A specialized component utilizing Twilio Functions and optical character recognition. This component makes it easy to parse purchase data from a reciept image sent to your Twilio phone number by utilizing an open source OCR API. Complete tutorial documentation can be found on this Google Docs page.

OC Habitats Web App
